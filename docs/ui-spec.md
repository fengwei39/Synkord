# Synkord UI 规范 v1.2

> 本文档约定 Synkord 前端的所有 UI 组件、页面结构、交互规范。详见 [requirements.md](./requirements.md) 的产品定位。

---

## 〇、本版本变更（相对 v1.1）

| 变更 | 说明 |
|---|---|
| 删除「团队」「成员」顶级 tab | 团队概念删除 |
| 删除原"切项目 confirm Modal" | 改为契约集切换下拉，无 confirm |
| `/mcp` 升级为顶级路由 | 不再是项目内的二级页面 |
| 新增「设为活跃」按钮 | 在契约集详情页 |
| 成员管理页面 | `/contracts/:id/members` |
| 顶栏 chip | 从"显示当前项目"升级为"可点击的切换下拉" |

## 一、页面清单

| 路径 | 页面组件 | 关键功能 |
|---|---|---|
| `/login` | `Login` | 登录 |
| `/mcp` | `McpConsole` | MCP 主控台（登录后默认） |
| `/contracts` | `ContractList` | 契约集列表（含"新建契约集"弹窗入口） |
| `/contracts/:id` | `ContractDetail` | 契约集详情（Tabs：接口/模型/影响/成员/基本信息） |
| `/contracts/:id/apis` | `ContractApis` | 接口列表（可独立路由，也可作 Tab） |
| `/contracts/:id/apis/:apiId` | `ContractApiDetail` | 接口详情/编辑 |
| `/contracts/:id/models` | `ContractEntities` | 数据模型列表 |
| `/contracts/:id/models/:modelId` | `ContractEntityDetail` | 数据模型详情/编辑 |
| `/contracts/:id/members` | `ContractMembers` | 成员管理 |
| `/contracts/:id/import` | `ContractImport` | 导入 OpenAPI/Swagger |
| `/settings` | `Settings` | 个人设置 |

## 二、顶级导航

### 2.1 布局

```
┌────────────────────────────────────────────────────────────────────────────┐
│  Synkord  │  MCP★  │  契约集  │  设置  │           [切换契约集▾]  [👤]      │
└────────────────────────────────────────────────────────────────────────────┘
```

### 2.2 元素规范

| 元素 | 位置 | 行为 |
|---|---|---|
| Logo | 最左 | 点击回 `/mcp` |
| **MCP★** | 第 1 位 | 核心功能；旁有 `<McpStatusDot>` 健康状态指示点 |
| 契约集 | 第 2 位 | 跳 `/contracts` |
| 设置 | 第 3 位 | 跳 `/settings` |
| **切换契约集▾** | 右上 | 点击展开下拉（`<ContractSwitcher>` 组件） |
| 👤 用户菜单 | 最右 | 下拉：个人信息、登出 |

### 2.3 MCP Tab 上的状态指示

`<McpStatusDot>` 紧贴「MCP★」文字右侧，显示 4 种状态：

| 状态 | 颜色 | 触发条件 |
|---|---|---|
| 运行中 | 绿 `#52c41a` | state=running + 最近 1 分钟有 IDE 连接 |
| 空闲 | 黄 `#faad14` | state=running + 最近 5 分钟无 IDE 连接 |
| 异常 | 红 `#ff4d4f` | state=failed 或最近 1 分钟有错误 |
| 启动中 | 灰 `#d9d9d9` | state=starting / restarting |
| 未启动 | 灰 `#d9d9d9` | state=stopped |

鼠标悬停显示 Tooltip：状态文字描述。

## 三、契约集切换下拉（`<ContractSwitcher>`）

### 3.1 触发位置

- 顶栏右上 chip（`variant="topbar"`）：文本显示当前活跃契约集名称 + 下拉箭头
- MCP 页面「活跃契约集」卡内（`variant="mcp-page"`）：作为完整切换按钮

### 3.2 下拉结构

```
┌──────────────────────────────────────────────┐
│ 切换契约集                                ✕ │
├──────────────────────────────────────────────┤
│ 🔍 [搜索契约集名...]                          │
├──────────────────────────────────────────────┤
│ ✓ 订单平台                  backend  当前  │
│   用户中心                  web            │
│   数据分析平台              app            │
│   内部 API 网关             backend        │
│   订单平台 v2               backend        │
│   CMS 后台                  web            │
│   ... (最多 8 条，超出滚动)                  │
├──────────────────────────────────────────────┤
│ 📁 管理所有契约集                        →   │
└──────────────────────────────────────────────┘
```

### 3.3 交互规范

| 行为 | 规范 |
|---|---|
| 触发 | 点击顶栏 chip |
| 搜索 | 实时过滤，无需回车；自动 focus 输入框 |
| 当前项 | ✓ 图标 + "当前" 标签 + 浅色背景 |
| 选中 | 调 `setActiveContract(id)` → 跳 `/contracts/:id` → 关闭下拉 |
| 切换失败 | toast 错误信息，下拉保持打开 |
| 切换成功 | toast "已切换到 XXX，AI 下次查询将使用此契约集" |
| 键盘 ESC | 关闭下拉 |
| 点击外部 | 关闭下拉 |
| 无 confirm Modal | 纯导航，无副作用 |
| 无匹配 | 显示"未找到契约集" + "创建契约集" 按钮 |

### 3.4 数据来源

```typescript
// 组件内部
const { data: contracts } = useApi(() => listContracts({ limit: 200 }), {
  refreshOnFocus: true  // 窗口聚焦时刷新，保证新创建的契约集立刻可见
})
const { data: active } = useApi(() => getActiveContract())
```

### 4.2 区块规范

| 区块 | 组件 | 数据源 | 轮询 |
|---|---|---|---|
| 状态卡 | `<Card>` + `<Descriptions>` | `getMcpStatus()` | 3 秒 |
| 活跃契约集卡 | `<Card>` + `<ContractSwitcher variant="mcp-page">` | `getActiveContract()` + `listContracts()` | 无（事件驱动） |
| IDE 配置区 | `<Tabs>` + `<Input.TextArea readonly>` + `<Button>` | `getIdeConfig()` | 无 |
| 快速开始 | `<Steps>` 或静态文本 | - | - |
| 高级操作 | `<Button>` 组 | - | - |

### 4.3 启停交互

**停止按钮**（必弹 ConfirmModal）：

```
┌─────────────────────────────────────────┐
│ 停止 Synkord MCP？                       │
│                                         │
│ 停止后所有 IDE 连接将断开：              │
│ • Cursor on MacBook Pro (2 分钟前)       │
│ • Claude Desktop (3 分钟前)              │
│                                         │
│ AI 在 IDE 里的查询将失败，直到 MCP 重新  │
│ 启动。                                  │
│                                         │
│              [取消]  [确认停止]           │
└─────────────────────────────────────────┘
```

**重启按钮**：无 confirm（用户意图明确），先停后启。

### 4.4 复制按钮反馈

所有「复制」按钮点击后：
- 复制到剪贴板
- 按钮文字短暂变为"已复制 ✓"
- 2 秒后恢复

```
│                                                             │
```


### 7.2 邀请成员

点击「+ 邀请成员」弹出 Modal：

```
┌─────────────────────────────────────────┐
│ 邀请成员                              ✕ │
├─────────────────────────────────────────┤
│ 搜索用户                                │
│ 🔍 [用户名或邮箱...]                     │
│                                         │
│ 匹配结果：                              │
│ • david                                  │
│ • daniel                                 │
│                                         │
│ 角色：○ viewer  ● editor  ○ owner        │
│                                         │
│        [取消]            [邀请]          │
└─────────────────────────────────────────┘
```

**注意**：owner 角色**不可在邀请时直接赋予**，避免误操作导致多 owner。新成员只能被邀请为 viewer 或 editor。

### 7.3 修改角色

- 仅 owner 可操作
- 创建者的角色下拉菜单禁用，显示「🔒 创建者」
- 其他成员可改为 owner / editor / viewer

### 7.4 移除成员

- 仅 owner 可操作
- 点击「移除」弹 confirm："确认从订单平台移除 bob？他将失去所有访问权限。"
- 创建者不显示「移除」按钮

## 十、通用组件清单

### 10.1 组件

| 组件 | 用途 | 关键 props | 关键行为 |
|---|---|---|---|
| `<LoadState>` | 统一 loading/error/empty 三态 | `loading`, `error`, `data`, `isEmpty?`, `children` | loading: Skeleton；error: Alert + 重试；empty: Empty |
| `<ContractSwitcher>` | 契约集切换下拉 | `variant: 'topbar' \| 'mcp-page'` | 搜索 + 列表 + 当前项 + 管理入口 |
| `<McpStatusDot>` | MCP 状态指示点 | `status: McpStatus`, `size?` | 4 色 + Tooltip |
| `<EmptyState>` | 通用空状态 | `description`, `action?` | 描述 + CTA 按钮 |
| `<ApiCard>` | API 列表项 | `api: ApiDefinition` | 路径/方法/标签/描述 |
| `<EntityCard>` | 实体列表项 | `entity: EntityDefinition` | 名称/字段数/引用数 |
| `<ConfirmModal>` | 通用确认对话框 | `title`, `content`, `onOk`, `danger?` | 用于 MCP 启停等危险操作 |

### 10.2 Hooks

| Hook | 用途 |
|---|---|
| `useUnsavedGuard(isDirty)` | 脏检查 + 离开确认 + 浏览器关闭提示 |
| `useActiveContract()` | 读取活跃契约集 + 实时更新 |
| `useMcpStatus()` | 读取 MCP 状态 + 3 秒轮询 |
| `useDebounce(value, ms)` | 防抖（用于搜索） |

### 10.3 工具函数

| 函数 | 用途 |
|---|---|
| `formatRelative(date)` | 相对时间显示（「2 分钟前」） |
| `stateLabel(state)` | MCP 状态文字映射 |
| `stateColor(state)` | MCP 状态颜色映射 |
| `methodColor(method)` | HTTP 方法颜色（GET 蓝、POST 绿 等） |
| `copyToClipboard(text)` | 复制到剪贴板 + 反馈 toast |
