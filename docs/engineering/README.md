# 工程文档

开发流程按 **功能迭代** 推进，每个迭代独立可演示、可验收，完成后合入 `dev` 再开下一轮。

## 开发节奏

```
定义迭代范围 → Claude Code 实现 → fixtures 验收 → 合入 dev → 下一迭代
```
