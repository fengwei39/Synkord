# Synkord 品牌规范

Synkord 的标志由蓝色同步箭头和中心节点组成，表达 API 契约、团队协作和 IDE AI 上下文同步。

## 标志文件

| 用途 | 文件 |
|---|---|
| 纯图标 SVG | `assets/brand/synkord-icon.svg` |
| 亮色背景横排（透明底） | `assets/brand/synkord-logo-light.svg` |
| 暗色背景横排（透明底） | `assets/brand/synkord-logo-dark.svg` |
| 多尺寸 PNG | `assets/brand/png/` |
| Web favicon | `frontend/public/favicon.svg` |
| Electron 打包图标 | `frontend/build/icon.svg`、`frontend/build/icon.png` |

## 颜色

| Token | Hex | 用途 |
|---|---:|---|
| Synkord Blue | `#1267ff` | 图标高光、主品牌色 |
| Synkord Blue Deep | `#0047e5` | 图标渐变暗部 |
| Synkord Navy | `#02091f` | 暗色横排背景 |
| Ink | `#05091f` | 亮色背景文字 |
| White | `#ffffff` | 暗色背景文字 |

## 使用规则

- 亮色背景使用 `synkord-logo-light.svg`，暗色背景使用 `synkord-logo-dark.svg`；两者均为透明底，不内置背景色。
- 小尺寸入口、favicon、应用图标使用 `synkord-icon.svg` 或 `png/icon-*`。
- 标志四周至少保留图标中心圆直径等宽的安全区。
- 不要拉伸、旋转、改色、加描边或阴影。
- 横排标志最小显示宽度建议为 160 px；纯图标最小显示尺寸建议为 24 px。
