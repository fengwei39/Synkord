## 四、契约 Schema 规范（contract-v1）

### 4.1 顶层结构

```json
{
  "$schema": "https://synkord.dev/schema/contract-v1.json",
  "pack": "string",           // 契约包名，如 auth-pack
  "version": "string",        // semver，如 1.2.0
  "conventions": { ... },     // 可选，命名与通用约定
  "entities": { ... }         // 实体定义，key 为实体名
}
```

### 4.2 conventions

```json
{
  "id_type": "uuid",
  "naming": {
    "db": "snake_case",
    "api": "camelCase",
    "java": "PascalCase",
    "go": "PascalCase"
  },
  "timestamps": {
    "created_at": "datetime",
    "updated_at": "datetime"
  }
}
```

### 4.3 实体（entities.{Name}）

```json
{
  "table": "users",
  "fields": {
    "id":     { "type": "uuid", "primary": true },
    "email":  { "type": "string", "maxLength": 255, "unique": true },
    "status": { "type": "enum", "values": ["active", "suspended"] }
  },
  "relations": {
    "roles": {
      "type": "many-to-many",
      "target": "Role",
      "through": "user_roles"
    }
  }
}
```

### 4.4 字段类型（fields.{name}.type）

| type | 额外属性 |
|---|---|
| `uuid` | `primary`, `unique` |
| `string` | `maxLength`, `unique`, `primary` |
| `int` | `primary`, `unique` |
| `boolean` | — |
| `datetime` | — |
| `enum` | `values`（必填，字符串数组） |
| `json` | — |

### 4.5 关联类型（relations）

| type | 必填属性 |
|---|---|
| `many-to-one` | `target` |
| `one-to-many` | `target` |
| `many-to-many` | `target`, `through` |
| `one-to-one` | `target` |

## 五、examples/auth-pack.json 要求

必须包含：

| 实体 | 字段 | 关联 |
|---|---|---|
| **User** | id, email, status, created_at, updated_at | roles → Role (many-to-many, through: user_roles) |
| **Role** | id, name, created_at, updated_at | — |

- `pack`: `"auth-pack"`
- `version`: `"1.0.0"`
- `conventions` 含 naming 与 timestamps
- 必须通过 Schema 校验

## 六、Rust API 设计

```rust
// crates/contract-core/src/lib.rs

/// 从文件加载并校验契约包
pub fn load_contract_pack(path: &Path) -> Result<ContractPack, ContractError>;

/// 从 JSON 字符串校验契约包
pub fn validate_contract_json(json: &str) -> Result<ContractPack, ContractError>;

/// 契约包结构体（serde 反序列化）
pub struct ContractPack {
    pub pack: String,
    pub version: String,
    pub conventions: Option<Conventions>,
    pub entities: HashMap<String, Entity>,
}
```

### ContractError 变体

```rust
pub enum ContractError {
    Io(std::io::Error),
    Json(serde_json::Error),
    SchemaViolation { path: String, message: String },
}
```
